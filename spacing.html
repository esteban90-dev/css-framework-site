Spacing

Spacing classes are based on a rem (px) scale.



Margins



.m-auto
.m-0 (0 rem)
.m-1 (.25 rem)
.m-2 (.5 rem)
.m-3 (1 rem)
.m-4 (2 rem)
.m-5 (4 rem)
.m-6 (8 rem)

To apply a margin to only one side of an element, use the following prefixes:


  • top: .mt-*
  • right: .mr-*
  • bottom: .mb-*
  • left: .ml-*



Padding



.p-0 (0 rem)
.p-1 (.25 rem)
.p-2 (.5 rem)
.p-3 (1 rem)
.p-4 (2 rem)
.p-5 (4 rem)
.p-6 (8 rem)

To apply padding to only one side of an element, use the following prefixes:


  • top: .pt-*
  • right: .pr-*
  • bottom: .pb-*
  • left: .pl-*
