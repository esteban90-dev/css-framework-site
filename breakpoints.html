Breakpoints

    By default, there are 3 responsive breakpoints available:
  • mobile (sm): 576px
  • tablet (md): 768px
  • desktop (lg): 992px

To make a class only apply at a certain breakpoint, append _sm, _md, or _lg to the class name.


For example, consider this div: it starts out at full width when the screen size is less than the small breakpoint, and as the screen increases in size, the width is reduced at each breakpoint. Try resizing the screen to see the effect.


.col-12 .col-6_sm .col-3_md .col-1_lg


This is a block of text that turns red at each breakpoint.
