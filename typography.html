Typography

For simplicity, a native font stack is used. For each one of these font types, the browser will use the first one in the list that is installed on the user's device. This means that the font may change from one device to the next, but most systems have fonts installed that look good.



Sans Serif Fonts


-apple-system, BlinkMacSystemFont, avenir next, avenir, segoe ui, helvetica neue, helvetica, Ubuntu, roboto, noto, arial, sans-serif



Serif Fonts


Iowan Old Style, Apple Garamond, Baskerville, Times New Roman, Droid Serif, Times, Source Serif Pro, serif, Apple Color Emoji, Segoe UI Emoji, Segoe UI Symbol



Monospace Fonts


Menlo, Consolas, Monaco, Liberation Mono, Lucida Console, monospace



    Use these classes to change the font type:
  • .sans-serif
  • .serif
  • .mono
